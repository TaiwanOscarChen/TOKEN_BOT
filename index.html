完整啟用 API 服務步驟

  1. 步驟一：線上條款簽署
    前往 永豐金線上簽署中心，完成「API 電子交易風險預告書暨使用同意書」之簽署（證券與期貨帳戶需分別簽署）。 ⚠️ 簽署完成的時間必須「早於」API 測試時間，否則測試紀錄將被視為無效。
  2. 步驟二：申請 API 金鑰 (API Key)
    前往 永豐金 API 金鑰申請中心 申請您的 API Key 與 Secret Key。
  3. 步驟三：下載電子交易憑證
    至永豐金證券官網透過憑證管理工具，將您的下單交易憑證 (.pfx) 匯出保存至本地端。
  4. 步驟四：執行本工具進行 API 測試
    • 使用 Shioaji 標籤頁：輸入 API Key 等資訊，送出模擬下單委託。
    • 使用 T4 API 標籤頁：輸入身份證與帳號，送出驗證委託。
    ⚡ 本工具已完全自動化此步驟，免去您手動撰寫代碼或操作 Excel VBA 的繁瑣。
  5. 步驟五：等待審核與重新登入
    測試成功送出後，測試數據會自動上傳永豐金伺服器，一般在營業日 14:01 ~ 17:59 期間審核最快約 10 分鐘開通。 審核通過後，即可使用 `simulation=False` 連接正式交易環境下單！
